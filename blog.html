2017-03-30: Meeting an old bug from windows 8 again in Windows 10

After installing the Creators Update (1704) I again met a bug I once had with windows 8. Every time I woke up my system from sleep all windows were resized to 1024 Ã— 768 and stacked in the upper left corner.

Luckily I remembered where to find a solution. The solution was - like in my previous post - modifying something in the registry. I just had to edit the two keys PrimSurfSize.cx and PrimSurfSize.cy ... IN EVERY SUBFOLDER CALLED 00!!!!

I hate this database called registry because you never knows where to find a specific setting or how it's related to the installed programs. With Linux you can often know where to look if something is going wrong.

2017-03-26: Having trouble with Windows 10 File Explorer

God damn Windows 10. Fucking File Explorer Crashed every time because something in QuickAccessToolbar was broken.

Deleting the following registry resets the QuickAccessToolbar: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Ribbon\QatItems
